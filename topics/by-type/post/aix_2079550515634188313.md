## 原文

Samsung is reportedly targeting mass production of its CMM-D 3.0 memory module, upgraded from CXL 3.1 to CXL 3.2, by year-end 2026. This remains a reported roadmap rather than a formally confirmed production commitment.

Samsung has meanwhile demonstrated a second-generation CXL 3.2 CMM-DDR5 sample, although it has not disclosed a mass-production date.

The real significance is not that CXL replaces HBM. It inserts a scalable, lower-cost memory tier between host DDR and SSD, allowing AI servers to expand and pool memory without adding CPUs or GPUs purely to gain capacity. CXL 3.2 also introduces hardware-based hot-page monitoring to improve data placement.

Samsung separately demonstrated that a 1TB CXL memory pool retained about 92% of DRAM-level inference performance in an eight-GPU KV-cache workload. SK hynix-backed researchers’ correctly named ITME architecture achieved up to 35.7% higher throughput than CPU offloading when large context workloads exhausted host memory—an experimental peak, not a universal performance gain.

For investors, CXL potentially expands AI-server DRAM content while creating new demand for memory controllers, switches and system software. Commercial adoption still depends on platform support and interoperability across CPUs, GPUs and servers.
