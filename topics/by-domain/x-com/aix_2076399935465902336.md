## 原文

大家以為玻璃基板是在換材料。

等一下，不是。

它是在換掉 AI 封裝的成本函數。

Reticle limit 約 26×33mm，限制的是單次曝光，不是整個 package。台積電可以靠 stitching，把 CoWoS 從 3.5×、5.5×一路往 9.5×甚至更大推。

所以 CoWoS 不是做不下去。

問題是越做越貴。

當 GPU、HBM、I/O、RDL、電源全部塞進同一個 package，每多一塊，對位、翹曲、良率和晶圓利用率就一起惡化。

這時候，方形 panel 才開始有意義。

300mm wafer 面積約 70,686mm²；515×510mm panel 約 262,650mm²，是 3.72 倍。更重要的是，AI package 本身也是方形。封裝越大，圓形 wafer 邊緣浪費越誇張。

所以 CoPoS 的核心不是「玻璃比較先進」。

是方形 panel 對大型方形 package 的經濟學，可能比圓形 wafer 更合理。

但 CoPoS 也不是整張玻璃塞滿 GPU、不切直接出貨。

更可能的路線，是用 panel 完成大型 RDL、TGV、build-up 和封裝整合，最後再切成個別 package。

Panel 是製程載體，不一定是最後產品。

玻璃的優點是真的：低 CTE、高剛性、尺寸穩定，適合大尺寸 fine-line RDL。

但每個優點後面都有代價。

玻璃不容易翹，但會裂；表面平滑，但銅和 SiO₂ 不好黏；導熱比樹脂好，但絕對取代不了冷板和液冷。

所以玻璃不是解決問題。

它只是把 organic core 的翹曲問題，換成 TGV、附著、鍍銅、切割、搬運和 microcrack 問題。

而且 Glass core 不代表 ABF 消失。

真正的結構更可能是：

ABF build-up
＋ Glass core
＋ TGV
＋ Copper interconnect
＋ Fine-line RDL

玻璃取代的是中間 core，不是整個 substrate。味之素、欣興、南電、景碩未必直接出局，真正發生的是價值重新分配。

所以台股真正有機會的，也不是誰家倉庫裡有玻璃。

而是誰能讓一張大 panel 經過十幾道製程後，還是沒有裂、沒有翹、沒有 open、沒有 void，而且每一張都做得一樣。

鈦昇看 TGV 設備、正式 PO 和虧轉盈。

盟立看 handling、EFEM 和自動化認證。

群翊有既有獲利，玻璃是增量。

雷科題材彈性最大，但財報證據最少。

這四家公司根本不是同一種投資。

最後真正要看的，不是新聞標題，而是：

技術展示
→ Demo
→ Qualification
→ Vendor List
→ PO
→ Acceptance
→ Repeat Order
→ 營收
→ 毛利
→ EPS

加入聯盟不是訂單。

Demo 不是量產。

有營收也不代表現在的股價合理。

玻璃基板真正的終局，不是玻璃取代所有材料，也不是 CoPoS 一夜殺死 CoWoS。

而是 AI 封裝正在從「裝晶片的盒子」，變成整合運算、記憶體、電源和光通訊的系統平台。

方向是真的。

但從方向變成 EPS，中間還隔著認證、良率、成本、客戶和時間。

物理層不會說謊。

股價最會騙人的，就是把這五件事全部折疊成同一天。

明天發完整版。
