THE CXL MEMORY RACE: SAMSUNG AND SK HYNIX TARGET AI INFERENCE BOTTLENECKS

TLDR: As AI inference scales, memory capacity is becoming as important as raw GPU performance. Samsung and SK Hynix are pushing CXL to pool memory across servers and reduce the bottlenecks created by larger models and more simultaneous users.

Samsung built a 1TB CXL memory pool using an Nvidia RTX Pro 6000 Blackwell GPU, a CXL switch and its CMM-D memory.

In a single-GPU setup, performance was similar to DRAM. Across 8 GPUs, the CXL pool maintained about 92% of DRAM performance while providing substantially more memory capacity.

The advantage became clearer when KV cache demand exceeded a 512GB DRAM configuration. The DRAM system slowed as it recomputed cache, while the 1TB CXL pool absorbed more KV cache and maintained stable inference performance.

SK Hynix is moving in parallel. It unveiled a 256GB CMM-DDR5 module and demonstrated it inside a CXL pooled-memory server, alongside HBM, server DRAM, eSSD and SOCAMM2.

The next AI memory stack will not be HBM alone. It will be HBM for bandwidth, DRAM for local capacity, CXL for pooled expansion and SSDs for deeper storage.
