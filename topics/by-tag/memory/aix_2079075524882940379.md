CXL’S NEXT MEMORY LAYER: A CHEAPER WAY TO SCALE AI INFERENCE

TLDR: Samsung and SK Hynix are racing to turn CXL into a new memory layer between DRAM and storage, giving AI systems far more capacity at lower cost as token usage explodes.

HBM and DDR are fast, but their capacity is physically tied to GPUs and CPUs. Adding more memory often means adding more processors too, which quickly becomes expensive.

CXL changes that by allowing large pools of memory to sit outside the processor and connect through a high-speed interface. It is slower than directly attached memory, but much cheaper and easier to expand.

SK Hynix says its new architecture places CXL memory between DDR and SSD, improving AI inference efficiency by 35.7%. The system predicts which data will soon be needed and moves it from storage into faster memory before the processor asks for it.

Samsung says its CMM-D 2.0 CXL memory can deliver 92% of DDR5’s inference performance while offering much easier capacity expansion. It is also developing hybrid products that combine DRAM and NAND.

The market is still young, but the growth forecast is aggressive. Yole Group expects the CXL market to rise from $2.1 billion this year to $15.8 billion in 2028, an increase of more than sevenfold.

Goldman Sachs sees monthly AI token usage rising 24x by 2030. Every extra token adds to the amount of data AI systems need to hold and move, making memory an even bigger bottleneck.

Nvidia, Microsoft, Intel and AMD are already building CXL support into new chips and cloud infrastructure. Samsung and SK Hynix are now competing to control the memory architecture that sits beneath the next generation of AI inference.
